KEIJI 物販API検証

プライバシーポリシー 利用規約

概要

KEIJIは、物販業務における価格調査、商品情報確認、仕入れ判断の補助を目的としたAPI検証用ツールです。 Yahoo!ショッピングAPIなどから取得した商品情報を、社内検証および人間による判断補助に利用します。

検証する内容

  • 商品名、商品コード、カテゴリ、価格などの商品情報の確認
  • 仕入れ候補と販売候補の商品同定の補助
  • 価格差、利益見込み、レビュー要否を確認するためのローカルレポート作成
  • API取得結果と手元データの照合精度の検証

利用予定のAPI範囲

初期検証では、Yahoo!ショッピングAPIのうち商品情報、カテゴリ、ランキング、在庫や販売状態などの確認に関わる読み取り用途を中心に扱います。 注文API、お問い合わせ管理API、定期購入APIなど、追加申請やより強い管理が必要なAPIは初期検証の対象外です。

行わないこと

  • 商品の自動購入、注文確定、決済の実行
  • 出品、価格変更、在庫変更などの自動実行
  • ログイン、カート投入、チェックアウトの自動操作
  • 取得した情報を用いた無断の外部送信や第三者販売

利用予定の情報

商品名、価格、商品コード、カテゴリ、在庫や販売状態に関する公開またはAPI提供情報を、 仕入れ候補の確認、価格差の把握、商品同定の精度確認に利用します。

運用方針

KEIJIの出力は、最終判断を自動化するものではありません。 仕入れ、購入、販売、出品、価格変更など外部に影響する操作は、人間が内容を確認したうえで別途判断します。
