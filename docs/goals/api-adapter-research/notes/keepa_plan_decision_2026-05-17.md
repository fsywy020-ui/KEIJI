# Keepaプラン判断メモ — 2026-05-17

## 前提

- 想定取得対象: Amazonランキング1位〜20,000位程度の商品情報。
- 頻度: 週1〜2回。
- 概算: 商品詳細1件 = 約1 token。
- 注意: live更新、offer詳細、Buy Box詳細、seller詳細などを深く取る場合は追加tokenが必要になる可能性あり。

## 試算

| プラン | 月額 | 1回20,000商品の所要時間 | 判断 |
|---|---:|---:|---|
| 20 tokens/min | 49ユーロ | 約16.7時間 | 月間token量は足りるが、1回が長い |
| 60 tokens/min | 129ユーロ | 約5.6時間 | 初月検証に最適 |
| 250 tokens/min | 459ユーロ | 約1.3時間 | 本格運用・複数カテゴリ向け |
| 500 tokens/min | 879ユーロ | 約40分 | 現段階では過剰寄り |

## 判断

- 初月推奨: 60 tokens/min、月額129ユーロ。
- 節約優先: 20 tokens/min、月額49ユーロ。
- 本格運用: 250 tokens/min、月額459ユーロ。

## 日本円目安

1ユーロ約184円前後の概算。

- 49ユーロ: 約9,000円/月
- 129ユーロ: 約23,700円/月
- 459ユーロ: 約84,500円/月
- 879ユーロ: 約161,700円/月

## 契約前確認ポイント

- 月額料金。
- tokens/min。
- いつでもキャンセル可能か。
- 未使用tokenが1時間で失効する点。
- 取得対象カテゴリが増え、実質10万商品以上になる場合は250 tokens/min以上を検討。
